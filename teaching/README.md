# Teaching materials — M5 TimerCAM streaming

Documents for instructors and students using this repository’s **streaming** lesson.

| Document | Audience | Purpose |
|----------|----------|---------|
| [**INSTALL_SIMPLE.md**](INSTALL_SIMPLE.md) | Students | **Short checklist only** — minimal steps, no extra explanation |
| [**INSTALL.md**](INSTALL.md) | Students | Same setup with **detail** (script option, paths, why bundled libraries) |
| [**GUIDE_streaming_TimerCAM_end_to_end.md**](GUIDE_streaming_TimerCAM_end_to_end.md) | Students | LAN testing, security, ngrok, troubleshooting, repo map |
| [**LESSON_TimerCAM_Arduino_setup.md**](LESSON_TimerCAM_Arduino_setup.md) | Teachers / triage | Compile errors, core vs library skew, manual patches |
| [**TEACHER_lesson_smoothing_TimerCAM.md**](TEACHER_lesson_smoothing_TimerCAM.md) | Teachers | Classroom prep, Wi‑Fi pitfalls, lesson scope |

**Start here:** [`INSTALL_SIMPLE.md`](INSTALL_SIMPLE.md) for a fast path, or [`INSTALL.md`](INSTALL.md) if you want the full notes — then [`GUIDE_streaming_TimerCAM_end_to_end.md`](GUIDE_streaming_TimerCAM_end_to_end.md) for the browser test and optional ngrok.

Paths in those docs assume the **repository root** is the folder that contains `teaching/`, `m5cam_stream_server/`, `arduino_libraries/`, and `scripts/`.
